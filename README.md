# sns-component

## To install the bindings via Composer, add the following to composer.json:
```{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/shivraj-ib/sns-component"
    }
  ],
  "require": {
    "https://github.com/shivraj-ib/sns-component": "dev-main"
  }
}
```
